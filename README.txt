Stop It
=======
Simple email collector to use at confereces.

User enters their email into the text box and hits return. The prizes then flash until the user hits return again. The prize they won display fullscreen until the user or a new user hits return again.

##Features
- Date/time, user's email, and prize is stored on a database.
- Uses Font Awesome for icons.
-  Email validation
- Checks locally if user has already entered their email.
- Ability to set 'non-winnable' prizes.
